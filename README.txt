NEISS INJURY ANALYSIS
=====================

Repository: <https://github.com/dva023/neiss-injury-analysis/>

DESCRIPTION
----------
This project leverages the National Electronic Injury Surveillance System (NEISS) 
database to develop an injury severity prediction model. We combine machine learning 
and natural language processing (NLP) techniques to analyze unstructured medical 
narratives. The system requires running a workbook with a TabPy server connection.

INSTALLATION
-----------

Prerequisites:

1. Required Software
   * Tableau Desktop

2. Network Requirements
   * Verify TabPy server accessibility
   * Check VPN/firewall/proxy settings

Tableau-TabPy Connection Setup
----------------------------
1. Launch Tableau Desktop
2. Navigate: Help > Settings and Performance > Manage External Service Connection
3. Configure Connection:
   * Service: TabPy
   * Hostname: tabpy.ericy.me
   * Port: 9004
   * Authentication: Enable "Sign in with username and password"
     - Username: dva023
     - Password: YL8bar-_3.jXGFet
   * Verify connection using "Test Connection"
4. Save settings with "OK"
